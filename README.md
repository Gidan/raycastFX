# RaycastFX
Old school Raycast rendering using the JavaFX canvas.
https://www.youtube.com/watch?v=rKYuGVl0Swk

## Compile
```
mvn compile
```

## Run
```
mvn javafx:run
```

## Controls
| Key | Effect                   |
|-----|--------------------------|
| W   | move forward             |
| S   | move backward            |
| A   | strafe left              |
| D   | strafe right             |
| Q   | rotate view to the right |
| E   | rotate view to the left  |
